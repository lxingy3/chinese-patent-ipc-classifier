# Field Schema

## `patent_ipc_clean.csv`

| Field | Description |
| --- | --- |
| `patent_title` | Patent title from a public patent page |
| `patent_abstract` | Patent abstract from a public patent page |
| `ipc_level_1` | IPC section |
| `ipc_level_2` | IPC class |
| `ipc_level_3` | IPC subclass |
| `ipc_level_4` | IPC group-level code retained from the source workflow |
| `ipc_level_1_name` | Chinese IPC level 1 name |
| `ipc_level_2_name` | Chinese IPC level 2 name |
| `ipc_level_3_name` | Chinese IPC level 3 name |
| `ipc_level_4_name` | Chinese IPC level 4 name |
