# Data Notice

This dataset contains public patent metadata: title, abstract, IPC code levels, and IPC label names. It does not include browser caches, private notes, model checkpoints, or third-party PDFs.

Use cases:

- Chinese patent IPC classification
- Label-distribution analysis
- Supervised NLP baselines
- Patent-domain text modeling

Please respect the terms of the original public patent sources when redistributing modified copies or using the data commercially.
